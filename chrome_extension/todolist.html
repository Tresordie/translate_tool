🔗 同步 Google Calendar

📖 查看使用指南 ▶

方式一：手动粘贴 Token（推荐）

1. 打开 Google OAuth Playground
2. 左侧选 Calendar API v3 → .../auth/calendar.events
3. 「Authorize APIs」→ 登录授权
4. 「Exchange authorization code」→ 复制 Access token
5. 粘贴到设置 → 点击「Google」同步

方式二：OAuth 自动授权

1. Google Cloud Console 创建 OAuth 客户端 ID
2. 重定向 URI: 当前页面地址
3. Client ID 填入设置 → 点击同步

🍎 一键导入 Apple 提醒事项

📖 查看使用指南 ▶

方式一：一键导入（需安装桥接器）

1. 终端运行一次：./install_url_scheme.sh
2. 点击工具栏 🍎 提醒 按钮
3. 自动触发 AppleScript，瞬间导入提醒事项 ✅

方式二：下载脚本文件

1. 点击 🍎 提醒，自动下载 .applescript 文件
2. 双击文件 → 脚本编辑器打开 → 点 ▶ 运行

📅 导入日历 (.ics)

📖 查看使用指南 ▶
1. 点击工具栏 .ics 按钮下载日历文件
2. 双击 .ics 文件导入到任意日历应用
3. 支持 Google Calendar、Apple Calendar、Outlook 等
支持 - [ ] - [x] * [ ] 语法
